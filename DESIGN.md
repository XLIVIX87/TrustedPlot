# TrustedPlot Design Context

## Product mood
- trust-first
- premium
- calm
- structured
- serious
- uncluttered

## Design references
- Stripe
- Linear
- Notion
- Airbnb trust/booking clarity

## UI priorities
1. Verification badges must be visible and meaningful
2. Listing cards must be clean and scannable
3. Admin and legal ops screens must be operationally efficient
4. Escrow and inspection timelines must feel structured and high-confidence
5. Document viewer must feel secure and deliberate

## Avoid
- noisy marketplace patterns
- ad-heavy layouts
- overly playful interactions
- dense unreadable admin tables
- "fake fintech" gradients and fluff

## Primary screens
- Homepage / discovery
- Listing detail
- Create listing flow
- Seller dashboard
- Verification queue
- Document viewer
- Inspection booking
- Escrow flow
- Admin dashboard
